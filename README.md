1.  **Engage**: The bot proactively asks 2 simple questions (Needs vs. Skills).
2.  **Logic**: It determines the user's intent (Medical Help vs. Logistics vs. Volunteering).
3.  **Automate**: It automatically routes the user to the correct form and pre-fills the category, reducing cognitive load by 40%.
